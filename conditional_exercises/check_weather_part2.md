### Solution
```JavaScript 
let weather = 'snowy';
switch (weather) {
  case 'sunny':
   console.log("It's a beautiful day!");
  break;
  case 'rainy':
   console.log("Grab your umbrella."); 
  break;
  case 'snowy':
   console.log("Stay inside, because it is snowing outside.");  
  break;
  case 'cloudy':
   console.log("It may rain today.");
  break;
  case 'cold':
   console.log("Grab your jacket.");
        break;
    default:
        console.log("It's night and dark.");
}
```
